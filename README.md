
# API 

## Quick start

Key dependencies:

- Node version 12.17.0
- Mongodb 4.2.6

Start development instance:

```bash
npm install

export PORT=2000
export NODE_ENV='development' # other options are 'staging', 'production'
npm run dev
```

Inject seed data:

```bash
npm run dev-seed
```

This will create a test user to allow login, user: `admin@test.com` and password: `pass123`
